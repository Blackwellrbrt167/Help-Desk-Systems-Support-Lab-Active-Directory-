Active Directory Domain Services Installation

Objective
Install the Active Directory Domain Services (AD DS) role on Windows Server to prepare the system for promotion to a domain controller.

Why This Matters
AD DS is the foundation of centralized identity, authentication, and authorization in enterprise Windows environments. Every domain lifecycle begins with this role installation.

Actions Performed
Verified Windows Server configuration including CPU, memory, disk, and networking
Opened Server Manager and launched Add Roles and Features Wizard
Selected the Active Directory Domain Services role
Installed required management tools and dependencies
Completed successful role installation

Validation
AD DS role visible in Server Manager
No installation errors reported
Server ready for post-deployment configuration

Skills Demonstrated
Windows Server administration
Role-based service installation
Foundational Active Directory knowledge
