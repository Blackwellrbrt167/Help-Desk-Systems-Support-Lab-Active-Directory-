Domain Controller Promotion

# Objective
Promote the Windows Server to a domain controller by creating a new Active Directory forest and domain.

# Why This Matters
Domain controllers provide authentication, authorization, and directory services for all domain-joined systems. Proper domain promotion is critical for secure enterprise environments.

# Actions Performed
Launched post-deployment configuration from Server Manager
Created a new forest with a custom domain name
Configured domain and forest functional levels
Set Directory Services Restore Mode password
Installed DNS services during promotion
Completed prerequisite checks and promotion

# Validation
Server rebooted successfully
Active Directory Users and Computers console available
Domain name resolves correctly via DNS

# Skills Demonstrated
Active Directory architecture
DNS integration with AD
Enterprise domain configuration
