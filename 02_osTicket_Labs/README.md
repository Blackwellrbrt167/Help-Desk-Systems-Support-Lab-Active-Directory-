# 02 osTicket Help Desk Labs

# Overview
This section documents the setup and configuration of an osTicket help desk system to simulate real service desk operations. The lab includes Ubuntu Server deployment, LAMP stack validation, osTicket installation, role-based access control, and evidence-based verification. The goal is to demonstrate job-ready help desk workflows such as ticket intake, triage, escalation, resolution, and documentation.
