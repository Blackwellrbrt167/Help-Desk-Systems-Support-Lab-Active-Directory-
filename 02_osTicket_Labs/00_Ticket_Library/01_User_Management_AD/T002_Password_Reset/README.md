# T002_Password_Reset

## Ticket Summary
- **Problem:**
    User Expereineced authentication isues after multiple unsuccessful login attempts.
- **Impact:**
    User was unable to access their domain profile and could not perform assigned work duties. 
- **Root Cause:**
    Repeated incorrect password entries caused authentication failure.
    Account was reviewed and confirmed **not locked** at the time of investigation.
- **Resolution:**
    Verified user identity per company policy. 
    Reviewed Active Directory account status 
    Reset user Password 
    Enforced password change at next logon 
    Instructured user to reset password within 24 hours 
    Confirmed successful access restoration 
- **Prevention:**
    Educated user on secure password handling practices 
    Advised use of the company-approved password manager 
    Reinforced avoiding repeated login attempts with outdated credentials
